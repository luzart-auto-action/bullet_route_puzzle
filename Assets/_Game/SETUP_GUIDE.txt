================================================================================
 BULLET ROUTE PUZZLE - SETUP GUIDE
 Version: 2.0 (voi Editor Tools)
 Unity: 2021.3+ (LTS recommended)
 Required Packages: DOTween (Free or Pro), TextMeshPro
================================================================================

WORKFLOW MOI: Su dung Editor Tools tu dong hoa gan het moi thu.
Menu: BulletRoute > ...


================================================================================
BUOC 1: CAI DAT PACKAGES
================================================================================

1. Import DOTween:
   - Asset Store > DOTween (Free) > Import
   - Tools > Demigiant > DOTween Utility Panel > "Setup DOTween"

2. Import TextMeshPro:
   - Window > TextMeshPro > Import TMP Essential Resources


================================================================================
BUOC 2: SETUP SCENE (1 CLICK)
================================================================================

Menu: BulletRoute > Scene Setup Wizard

Bam "SETUP FULL SCENE" -> Tu dong tao:
  - 10 Manager GameObjects (GameManager, GridManager, TileFactory, ...)
  - Main Camera + CameraController
  - Grid Collider (layer "Grid" tu dong tao)
  - UI Canvas + GameplayUI + WinPanel + FailPanel
  - EventSystem, Directional Light
  - Tu dong gan references (camera, layers, UI panels, buttons)
  - Tu dong tao GameConfig ScriptableObject

Neu muon setup tung phan: bam cac nut rieng trong Wizard.


================================================================================
BUOC 3: TAO TILE PREFABS (1 CLICK)
================================================================================

Menu: BulletRoute > Tile Prefab Creator

Option A: Bam "TAO TAT CA 11 TILE PREFABS"
  -> Tao 11 prefabs voi placeholder 3D models
  -> Tu dong gan vao TileFactory

Option B: Bam tung nut de tao tung tile rieng
  -> Moi nut tao 1 prefab voi dung hierarchy:
     Tile_[Type]
       VisualRoot (target animation)
         Model (placeholder 3D)
       FX_Center, FX_Top, FX_Bottom, FX_Left, FX_Right
  -> Script, references, FX points tu dong gan

Sau khi tao xong: thay the placeholder models bang art cua ban.
Chi can sua trong Prefab > VisualRoot > thay model.


================================================================================
BUOC 4: TAO BULLET PREFAB (1 CLICK)
================================================================================

Menu: BulletRoute > Create Bullet Prefab

-> Tao Bullet.prefab voi:
   - BulletController script
   - VisualRoot > Sphere model + Glow
   - TrailRenderer
   - FX_Front, FX_Center, FX_Trail
   - Tu dong gan vao BulletManager


================================================================================
BUOC 5: THIET KE LEVELS (VISUAL EDITOR)
================================================================================

Menu: BulletRoute > Level Editor

Cach dung:
  1. Bam "New" de tao LevelData moi (hoac keo asset co san vao)
  2. Dat Grid Width x Height, star thresholds
  3. Chon Paint Mode:
     - Tile: chon type + rotation, click vao grid de dat
     - Turret: chon fire direction, click de dat
     - Target: click de dat
     - Eraser: click de xoa
  4. Click tren grid de paint
  5. Bam "Gan LevelData nay vao LevelManager"

Tips:
  - Bam "Dien Straight vao o trong" de fill grid nhanh
  - Portal can DAT 2 TILES CO CUNG Portal ID
  - Mirror: ExtraData 0 = (/), ExtraData 1 = (\)
  - Xem Summary de check so luong tiles/turrets/targets


================================================================================
BUOC 6: VALIDATE PROJECT
================================================================================

Menu: BulletRoute > Project Validator

Bam "RE-VALIDATE" -> Kiem tra tat ca:
  - 10 Managers co trong scene khong
  - Camera + CameraController
  - Grid Collider + layer
  - TileFactory co du 11 prefabs
  - LevelManager co levels
  - Bullet prefab
  - UI panels + dung ten
  - FX entries
  - Audio entries

Click vao ket qua de nhay den object loi.


================================================================================
BUOC 7: THEM ART (THU CONG)
================================================================================

Thay the placeholder 3D models trong prefabs:
  1. Mo prefab (double-click)
  2. Xoa placeholder model trong VisualRoot
  3. Keo 3D model moi vao VisualRoot
  4. Dam bao model nam o center, scale vua voi cell size (~1.0)
  5. Save prefab

FX Particles:
  1. Tao ParticleSystem prefabs trong Assets/_Game/Prefabs/FX/
  2. Chon [FXManager] trong scene
  3. Them entry: Name (chinh xac) + Prefab + Pool Size + Lifetime

Audio:
  1. Import audio clips vao project
  2. Chon [AudioManager] trong scene
  3. Them entry vao SFX Library hoac Music Library

================================================================================
TILE ROUTING LOGIC (de thiet ke level)
================================================================================

  Straight (Rot 0): Up <-> Down
  Straight (Rot 1): Left <-> Right
  Corner (Rot 0):   Up <-> Right
  Corner (Rot 1):   Right <-> Down
  Corner (Rot 2):   Down <-> Left
  Corner (Rot 3):   Left <-> Up
  Cross:            Xuyen thang qua moi huong
  Block:            Chan dan
  Mirror (/):       Up<->Right, Down<->Left
  Mirror (\):       Up<->Left, Down<->Right
  Splitter:         Tach dan 2 huong vuong goc
  Portal:           Teleport den portal cung ID, giu huong
  Bomb:             Xuyen qua, pha Block ke ben
  Absorb:           Nuot dan -> fail


================================================================================
WORKFLOW TOM TAT
================================================================================

  1. Import DOTween + TextMeshPro
  2. BulletRoute > Scene Setup Wizard > SETUP FULL SCENE
  3. BulletRoute > Tile Prefab Creator > TAO TAT CA 11 TILE PREFABS
  4. BulletRoute > Create Bullet Prefab
  5. BulletRoute > Level Editor > thiet ke levels
  6. BulletRoute > Project Validator > kiem tra
  7. Thay art, them FX, them audio
  8. Play!
